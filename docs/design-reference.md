# AtlasMe Design Reference

## Canonical Prototype

Use the prototype image shared in chat on 2026-05-06 as the primary visual reference for this repo.

This reference includes:

- Dark mode row: `Home`, `Journeys`, `Map`, `Explore`, `Profile`
- Light mode row: `Home`, `Journeys`, `Map`, `Explore`, `Profile`
- Secondary flows on the right: `Add Journey`, `Badge Wall`, `Globe Style`

## Alignment Rules

- Prefer the prototype's information architecture over the current implementation when they differ.
- Match tab naming to the prototype: `Home`, `Journeys`, `Map`, `Explore`, `Profile`.
- Treat the dark and light themes in the prototype as the target visual system.
- Use the prototype's spacing, card density, icon choices, and typography hierarchy as the source of truth.
- For the home screen, prioritize matching the hero title, globe treatment, stats row, recent journey card, and bottom tab bar.

## Important Differences Already Observed

- Current home hero text does not match the prototype exactly.
- Current globe rendering is more abstract than the prototype.
- Current top-right home action differs from the prototype.
- Current bottom tab labels differ from the prototype in places.
